# iPix-flutterapp

Wallpaper app by Flutter

# Installation

1. Clone this repo

2. Go to Home directory
  ```bash
  cd iPix
  ```
  
3. Add [pexels](https://www.pexels.com/api/?locale=en-US) APIKEY to  /lib/data/data.dart

4. Connect your device or emulator

5. Run this command
  ```bash
  flutter run --verbose
  ```
